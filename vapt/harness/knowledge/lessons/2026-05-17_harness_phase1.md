# 2026-05-17 Harness Phase 1 Lessons

- Unsafe proof execution and weak deduplication damage credibility fastest.
- Source-pattern systems must have one config and a regression corpus.
- A fresh model session needs a single cold-start command, not scattered docs.
- Candidate transitions should be enforced by the harness instead of remembered
  by the operator.
- Realtime authorization drift needs REST-denied negative controls.
