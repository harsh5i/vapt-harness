# Harness Corpus

This folder is the operator-local home of the cross-engagement
ledgers and tuning artifacts:

- `candidates.jsonl` — the canonical candidate ledger across all
  engagements on this machine. Gitignored.
- `submissions.jsonl` — the canonical submission ledger across all
  engagements on this machine. Gitignored.
- `step_outcomes.jsonl` — runtime telemetry from the orient/submit
  binding loop. Gitignored.
- `outcome_tuning.{yaml,md}` — generated by `outcome-tune`.
  Gitignored.

Only this README travels with the repo. The harness creates the
ledger files on first write.
